# QA-Genius AI

AI-Powered Test Design & Automation Assistant

## Overview

QA-Genius AI transforms software requirements into:

- Test Cases
- API Test Scenarios
- Risk Analysis
- Automation Scripts

Supported Frameworks:

- Playwright
- Cypress
- Selenium

## Tech Stack

- Python
- Streamlit
- Google Gemini API

## Features

### Generate Test Cases

- Positive Test Cases
- Negative Test Cases
- Edge Cases
- Security Test Cases

### Generate Automation Scripts

- Playwright
- Cypress
- Selenium

### Generate API Scenarios

- Request Validation
- Response Validation
- Error Handling

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
