# QA-Genius AI

AI-Powered Test Design & Automation Assistant

## Project Overview:

QA-Genius AI is an AI-powered test engineering platform developed to automate and enhance software quality assurance processes. The application leverages Google's Gemini large language model to transform software requirements and execution logs into actionable QA artifacts.

The platform assists QA engineers by generating comprehensive test cases, automation scripts, and intelligent root-cause analysis reports, reducing manual effort and improving testing efficiency.

The project was developed using Python, Streamlit, and the Gemini API, with the goal of exploring practical applications of Artificial Intelligence in software engineering and test automation.

QA-Genius AI transforms software requirements into:

- Test Cases
- API Test Scenarios
- Risk Analysis
- Automation Scripts

Supported Frameworks:

- Playwright
- Cypress
- Selenium
